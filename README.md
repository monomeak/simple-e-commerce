# Simple Clothing Store

A simple e-commerce website for learning basic HTML, CSS, and JavaScript.

## Features

- Shows a list of clothing products
- Displays product images, names, and prices
- Adds products to the shopping cart
- Updates total items and total price

## Folder Structure

```txt
simple-e-commerce/
├── index.html
├── style.css
├── script.js
└── README.md
```

## How To Run

Open `index.html` in your web browser.

## What You Learn

- HTML page structure
- CSS layout and design
- JavaScript arrays
- DOM manipulation
- Simple cart logic
